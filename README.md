
# TruckStacker

Tauri + Vue 3 + Vite製のデスクトップアプリです。

## 概要
- トラック荷台（長方形）と音響機器（SVG多角形）を管理・配置するシミュレーター
- 荷台・機器の登録/削除、SVGドラッグ＆ドロップ、回転・複製機能

## セットアップ
```
npm install
npx tauri dev
```

## 主な機能
- 荷台の登録・削除・選択
- 音響機器の登録・削除・複製・回転・配置（SVG）
- データはローカルJSONで管理

## 開発
- `src/` 以下にVueコンポーネントを配置
- Tauriバックエンドは`src-tauri/`に配置
